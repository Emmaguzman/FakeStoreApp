Desafío de React: Aplicación de CRUD de Productos
Descripción del Proyecto
el desafio consistira en la creacion de una aplicación web utilizando
 React que permita a los usuarios realizar operaciones CRUD. 

Productos utilizando una API existente. Deberás utilizar TypeScript para escribir el código,
 Redux Toolkit Query para manejar las solicitudes HTTP,
  Material UI para el diseño de la interfaz de usuario,
   Styled Components para personalizar el estilo, y
    Formik para manejar formularios y validación de datos.

Requisitos Funcionales
Listado de Productos: Los usuarios deberían poder ver un listado de productos existentes. Cada producto debe mostrar su nombre, descripción, precio y cualquier otra información relevante.

Validación de Formularios: Debe utilizarse Formik para manejar la entrada de datos en los formularios, incluyendo la validación de campos obligatorios y formatos de datos válidos.

Detalle del Producto: Los usuarios deberían poder ver los detalles completos de un producto, incluyendo su nombre, descripción, precio y cualquier otra información relevante.

Agregar Producto: Los usuarios deberían poder agregar un nuevo producto proporcionando su nombre, descripción, precio y cualquier otra información necesaria.

Requisitos Técnicos
Utiliza Redux Toolkit Query para manejar las solicitudes HTTP a la API existente y gestionar el estado de la aplicación de manera eficiente.

Implementa el diseño de la interfaz de usuario utilizando los componentes proporcionados por Material UI y personaliza el estilo utilizando Styled Components según sea necesario.

Utiliza TypeScript para escribir el código y asegurarte de que la aplicación sea robusta y fácil de mantener.

Recursos
La API existente proporciona endpoints para realizar operaciones CRUD en productos. La documentación de la API se puede encontrar en https://fakestoreapi.com/docs , https://jsonplaceholder.typicode.com/guide/ , https://dummyjson.com  debe seleccionar alguna de estas
Utiliza los recursos proporcionados por Redux Toolkit Query, Material UI, Styled Components y Formik para implementar las funcionalidades requeridas.
Entrega
Sube tu código a un repositorio de GitHub y proporciona el enlace al mismo.
Proporciona instrucciones claras sobre cómo ejecutar y probar la aplicación localmente.
